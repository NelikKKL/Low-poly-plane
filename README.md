# Low poly planes
это простая offline HTML игра
в игре нужно уворачиваться от препятсвий, на данный момент игра в разработке, и может работать не стабильно

если вы не хотите собирать игру то скачайте из репризитория plane-single.html

### музыка
вся музыка в игре не имеет АП

### сборка ⛳
для сборки скопируй репризиторий через:

`git clone https://github.com/NelikKKL/Low-poly-plane`

зайди в дерикторию, далее `python build.py`
на выходе вы получите единый html который можно запустить на любом устройстве без локального сервер
